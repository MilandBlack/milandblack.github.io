Firewall Rules for DMZ w/ Web Server

DMZ -> LAN: [DMZ IPs] ANY ALLOW: Applications


DMZ -> EXT: ANY ANY ALLOW: Web Browsing


LAN -> LAN: ANY ANY ALLOW


LAN -> EXT: ANY ANY ALLOW: Web Browsing


LAN -> EXT: ANY ANY DENY: P2P, torrent


EXT -> LAN: ANY ANY DENY


EXT -> DMZ: ANY ANY ALLOW: Web Browsing, Applications
