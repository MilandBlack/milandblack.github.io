Steps for Success

1. Change your password


2. Remove unnecessary programs and services


3. Update your system


4. Install nmap and run the following command: nmap -sV --script vulners [Network_IP/subnetLength]


5. Patch any found vulnerabilities


6. Install Wireshark and use filters to identify reconnaissance


Do as many injects as possible!

Prioritize!

These steps should set you on a good path. After these steps are implemented, be sure to complete as many injections as possible using the time alottments as a priority scale.
